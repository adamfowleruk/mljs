Class: docheadviewer

docheadviewer

new docheadviewer(container)

Views the head section properties from an XHTML document. These properties are always embedded within a document. This is distinct from MarkLogic document properties which are held in the document properties fragment and are nothing to do with XHTML.

Parameters:
Name Type Description
container string

ID of the HTML element to draw this widget in
